desert & sbr test Randomiser is a tool used to create randomised selections of Stands, Specs and Horses for desert and sbr test.

Use the 'Select All' button to generate all of them. Press the one for the game that you are playing.

Using the alternative set disables some Stands, Specs and Horses. For more information, see the list of changes.
